# Ultimate Tic Tac Toe
An AI agent that plays Ultimate Tic Tac Toe using Monte Carlo Tree Search.
## Overview

Ultimate Tic Tac Toe is an advanced version of the classic Tic Tac Toe game. It consists of a 3x3 grid of Tic Tac Toe boards, where each move you make determines the board on which your opponent must play.

## How to Play

1. The game is played on a 3x3 grid of 3x3 grids.
2. The first player can play anywhere on the board.
3. The position of the move determines the board on which the next player must play.
4. The game is won by getting three in a row on a board.
5. To win the game, you must win three boards in a row.
6. If a board is full, the next player can play anywhere on the board.
7. If a board is won or full, the next player can play anywhere on the board.
8. If a player is forced to play on a full or won board, they can play anywhere on the board.
9. If a player is forced to play on a full or won board, they can play anywhere on the board.
10. The game is a draw if all boards are full and no player has won.
